## 1. O que e um objeto em programacao?
Um objeto é uma entidade que guarda dados e pode executar ações.

## 2. O que e uma propriedade de um objeto? Cite um exemplo.
Uma propriedade é uma característica do objeto. Exemplo: um carro pode ter a propriedade cor = "vermelho".

## 3. O que e um metodo? Qual a diferenca entre metodo e propriedade?
Um método é uma função que pertence ao objeto. A diferença é que propriedade guarda informações e método executa uma ação.

## 4. O que e uma classe?
Uma classe é um modelo usado para criar objetos.

## 5. Qual a diferenca entre classe e objeto?
A classe é o modelo; o objeto é a instância criada a partir desse modelo.

## 6. Para que serve o constructor dentro de uma classe?
O constructor serve para inicializar o objeto, definindo os valores das propriedades quando ele é criado.

## 7. O que significa a palavra this dentro de uma classe?
A palavra this representa o próprio objeto atual da classe.

## 8. O que a palavra new faz ao criar um objeto?
A palavra new cria uma nova instância de uma classe, ou seja, um novo objeto.

## 9. Em TypeScript, por que usamos tipos como string, number e boolean?
Usamos tipos como string, number e boolean para definir o tipo dos dados, evitando erros e facilitando o desenvolvimento.
