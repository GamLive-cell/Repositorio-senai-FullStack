## 1. O que é programação?
É o processo de criar instruções para que um computador execute tarefas e resolva problemas.

## 2. O que é um algoritmo?
É uma sequência organizada de passos usada para alcançar um objetivo ou resolver um problema.

## 3. Qual a diferença entre algoritmo e linguagem de programação?
O algoritmo descreve a lógica da solução. A linguagem de programação é a forma de escrever essa solução para que o computador a execute.

## 4. O que é JavaScript?
JavaScript é uma linguagem de programação usada para criar funcionalidades e interatividade em sites e aplicações.

## 5. Onde o JavaScript pode ser utilizado atualmente?
Pode ser usado em sites, aplicações web, servidores, aplicativos móveis.

## 6. O que é o Node.js?
Node.js é um ambiente que permite executar código JavaScript fora do navegador.

## 7. Qual é a função do Node.js no desenvolvimento com JavaScript?
Ele permite criar aplicações de servidor, APIs, automações e outras soluções usando JavaScript no backend.

## 8. O que é o VS Code?
Ele permite criar aplicações de servidor, APIs, automações e outras soluções usando JavaScript no backend.

## 9. Qual a diferença entre um editor de texto comum e um editor de código
Um editor de código possui recursos específicos para programação, como destaque de sintaxe, autocompletar e depuração.

## 10. O que é o terminal?
O terminal é uma ferramenta que permite executar comandos e interagir com o sistema operacional por texto.

## 11. Para que serve o comando Node arquivo.js?
Esse comando executa o arquivo JavaScript chamado arquivo.js usando o Node.js.

## 12. O que é uma variável?
Uma variável é um espaço na memória usado para armazenar dados que podem ser utilizados no programa.

## 13. Qual a diferença entre Let e Const?
let permite alterar o valor da variável após sua criação. const cria uma variável cujo valor não pode ser reatribuído.

## 14. O que são tipos de dados? Cite exemplos.
Tipos de dados definem o tipo de informação armazenada. Exemplos: texto (string), número (number), verdadeiro ou falso (boolean), lista (array) e objeto (object).

## 15. Qual principal diferença entre JavaScript e TypeScript?
JavaScript é uma linguagem sem tipagem estática. TypeScript é uma extensão do JavaScript que adiciona tipagem e recursos extras para facilitar o desenvolvimento e reduzir erros.
