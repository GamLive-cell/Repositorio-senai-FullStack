## 1. O que é um array em programação?
Um array é uma forma de guardar vários valores dentro de uma única variável. É uma lista.

## 2. Por que é melhor usar um array do que criar várias variáveis separadas, como aluno1, aluno2 e aluno3?
Porque fica mais organizado e facilita trabalhar com muitos dados ao mesmo tempo.

## 3. O que é índice em um array?
Índice é a posição de um elemento dentro do array.

## 4. Por que o primeiro elemento de um array fica na posição 0?
Porque as linguagens de programação normalmente começam a contar a partir do zero.

## 5. Para que serve o .length?
O .length serve para mostrar quantos elementos existem no array.

## 6. Como acessamos o primeiro elemento de um array chamado alunos?
Usamos alunos[0] para acessar o primeiro elemento.

## 7. Como podemos alterar um valor dentro de um array?
Basta informar a posição e colocar um novo valor nela.

## 8. Qual a relação entre arrays e laços de repetição?
Os laços de repetição ajudam a passar por todos os elementos do array automaticamente.

## 9. Qual a diferença entre percorrer um array com for e apenas mostrar um item específico? 
Com o for percorremos todos os itens; ao acessar um índice específico, vemos apenas um item.

## 10. O que é um acumulador? Dê um exemplo de uso com notas ou valores.
Acumulador é uma variável usada para guardar a soma ou o resultado de vários valores. Exemplo: somar todas as notas de uma turma.

## 11. Para que servem operadores aritméticos como +, -, * e / dentro de um programa?
Eles servem para fazer cálculos, como somar, subtrair, multiplicar e dividir.

## 12. Para que servem operadores lógicos como && e ||?
Eles servem para combinar condições em uma decisão do programa.

## 13. Qual a diferença entre if/else e switch?
O if/else é usado para condições variadas; o switch é mais prático quando há várias opções para um mesmo valor.

## 14. O que os métodos push e pop fazem em um array?
O push adiciona um item ao final do array e o pop remove o último item.

## 15. O que é uma matriz e como ela se relaciona com arrays?
Uma matriz é um conjunto de arrays dentro de outro array, formando linhas e colunas.
