## Pesquisa - Lacos de Repetição..

## 1. O que e uma estrutura de repeticao?
Respota: É um recurso da programação que permite repetir um bloco de código várias vezes enquanto uma condição for verdadeira ou até ela ser atingida.
## 2. Para que serve o laco for?
Respota: Serve para repetir um bloco de código um número definido de vezes, geralmente quando já sabemos o limite de repetições.
## 3. Para que serve o laco while?
Respota: Serve para repetir um bloco de código enquanto uma condição for verdadeira, sem precisar saber exatamente quantas vezes vai repetir.
## 4. Para que serve o laco do-while?
Respota: Serve para executar um bloco de código pelo menos uma vez e depois repetir enquanto a condição for verdadeira.
## 5. Qual e a diferenca entre for, while e do-while?
Respota: O for é usado quando sabemos o número de repetições.
O while verifica a condição antes de executar o código.
O do-while executa primeiro e só depois verifica a condição, garantindo pelo menos uma execução.
